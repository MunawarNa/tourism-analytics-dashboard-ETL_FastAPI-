# Tourism Analytics Dashboard

## Description

**Tourism Analytics Dashboard** is a full-stack web application that visualizes hotel data for tourism analysis. The project combines **data engineering**, **Python backend development**, and a **React frontend** to provide interactive and insightful visualizations for tourism data.

The application fetches real-time hotel data from a **public Hotels API**, performs **ETL (Extract, Transform, Load)** operations, stores it in a **PostgreSQL database**, and exposes the data through a **FastAPI backend**. The frontend, built with **React and Chart.js**, displays interactive charts including:

- Total hotels per city  
- Hotel rating distribution  
  

The dashboard is designed to be **responsive, compact, and user-friendly**, allowing decision-makers or users to quickly analyze tourism trends.

> **Note:** This project focuses on the **data engineering pipeline and Python backend development**.  
> The frontend/dashboard was generated using AI for demonstration purposes and is included to visualize the backend data.
